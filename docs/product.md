# Product

## 目的

異なる GitHub リポジトリ間でファイルを自由に比較できる Web ツール。
通常の diff ツールでは対象にならない「まったく関係のないリポジトリのファイル」も比較対象にできる。

## 対象ユーザー

- 複数リポジトリにわたってコードを管理・レビューするエンジニア
- OSS と自社コードの差異を調べたいエンジニア
- ブランチや組織をまたいだファイル比較が必要な開発者

## 成功指標

- トークンなしでパブリックリポジトリのファイルを比較できる
- PAT を入力することでプライベートリポジトリのファイルも比較できる
- 比較結果をサイドバイサイド / ユニファイドの両形式で表示できる
- GitHub Pages に静的サイトとしてデプロイできる

## セキュリティ考慮

- PAT はブラウザの `localStorage` に保存する。静的サイト（サーバーサイド処理なし）・外部スクリプト未使用・ユーザー入力を HTML レンダリングしない構成のため、XSS 経由の漏洩リスクは限定的。開発者向けツールとして許容範囲とする
- 共有端末での利用時は設定モーダルの「削除」で PAT をクリアできる
